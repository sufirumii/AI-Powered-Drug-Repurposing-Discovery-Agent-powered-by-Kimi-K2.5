# AI Drug Repurposing Discovery Agent Prototype powered by Kimi K2.5 by MoonShotAI

Single-file HTML web app that uses AI to suggest FDA-approved drugs for potential repurposing in rare and orphan diseases.

### Features
- Disease input + options for analysis depth, focus area, database scope  
- Returns 4–6 ranked drug candidates (confidence, mechanism, evidence, safety, timeline)  
- Eight interactive Chart.js visualizations  
- Modern glassmorphic dark UI with animated particles  
- Mock data fallback when API is unavailable  

### Technology
- HTML + CSS (glassmorphism styling)  
- Vanilla JavaScript  
- Chart.js v4.4.0  
- Hugging Face Inference API  
- Model: moonshotai/Kimi-K2.5
